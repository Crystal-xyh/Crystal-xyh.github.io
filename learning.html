Experience of web design

During this semester, I have learned a lot knowledge of web design with friendly Human Computer Interaction, as well as HTML and CSS coding from lectures, MOOC and C3School website.

At the beginning, I really curious about this course, as although I am able to build website via a web hosting service provider, like WordPress, I do not understand website coding. When I start to design my own website, the only thing I can do is looking for some HTML and CSS code in the internet, and imitate these samples. But, in the process of the imitation, I find that my website has its special style which is different from other website. This means I have developed the abilities of creation and flexibility during my learning and imitating.

However, at the middle stage of my web design, I realized that I have utilized a lot of "div" tags without id or class, which always confused me, as it is hard to make sure which block I was styling. Besides, another problem I met during my design is that I have not paid any attention on different website style for different devices. When I watched my website on the iPad or iPhone, it was in a mess.

Therefore, in the future, I will keep developing my website. I believe that I will do a better job on web design and some creative things may appear in my own website after I do more practice.
